# archipelago2092.xyz

This is the source code of [archipelago2092.xyz](https://archipelago2092.xyz) website.

## build, run, contribute

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
